FIREBASE HOSTING (quello che tu chiami "fire bird")

Guida ufficiale: Firebase Hosting quickstart (Firebase docs).

1) Installa Firebase CLI
   npm install -g firebase-tools

2) Login
   firebase login

3) Nel tuo progetto (cartella della repo), inizializza hosting
   firebase init hosting

   - Seleziona il progetto Firebase
   - "public directory": metti la cartella che contiene index.html
     (se il tuo index.html è in root, puoi usare: .  oppure crea /public e sposta tutto dentro)
   - SPA? se è single page, metti YES

4) Deploy
   firebase deploy

Ti darà due URL:
- https://<project>.web.app
- https://<project>.firebaseapp.com

NOTE:
- Se usi Firebase Auth/Firestore/Storage, NON serve backend.
- Caricamento locandine: apri il sito -> Admin -> login -> upload.
